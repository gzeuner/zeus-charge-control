APIs and Data Sources

This application supports the following APIs: Sonnen API v2, aWATTar API v1, Tibber API (beta), and the Open-Meteo Weather API.

If using the Open-Meteo API, please note that it is free only for non-commercial use. For commercial usage, an API key is required. For more details, see the Open-Meteo Terms of Use.
