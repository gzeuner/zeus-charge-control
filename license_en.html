APIs and Data Sources

The software supports the Sonnen API v2, aWATTar API v1, Tibber API (Beta), and weather data from the Open-Meteo API.

Please note that the Open-Meteo API is free for non-commercial use only. For commercial use, an API key is required. For more information, see the Open-Meteo Terms of Use.
